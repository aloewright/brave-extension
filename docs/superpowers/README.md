# Superpowers Archive

This directory contains historical plans and specs. Some older entries mention
the previous Plasmo-based extension architecture.

Those references are archival only. The current extension build path is the
custom MV3 builder in `scripts/build-extension.mjs`, invoked by `pnpm build`
and `pnpm build:extension`; the loadable unpacked extension is `build/`.

Current architecture notes live in `AGENTS.md`; prefer the live source tree and
tests over older plans when they disagree.
